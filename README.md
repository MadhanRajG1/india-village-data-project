# 🇮🇳 India Village Data Project

## 📌 Overview
This project focuses on cleaning and combining village-level geographical data across multiple Indian states.

## 🔧 Tools Used
- Python (Pandas)
- Microsoft Excel
- VS Code

## ⚙️ Process
- Cleaned raw datasets from multiple states
- Standardized columns: State, District, SubDistrict, Village
- Removed duplicates and inconsistencies
- Automated merging using Python

## 📊 Output
A final dataset:
final_cleaned_india_data.csv

## 🚀 Key Learnings
- Handling messy real-world datasets
- Fixing encoding issues
- Automating multi-file processing
